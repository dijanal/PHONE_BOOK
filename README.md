## PhoneBook APP

 Basic web application for managing contacts online.

 Back-End: https://github.com/dijanal/PHONE_BOOK_API

## Build with

 * ReactJS
 * ExpressJS
 * SQLite3

## Installing

  Type in CMD:

     mkdir phonebook_app
     cd phonebook_app
     git clone 'https://github.com/dijanal/PHONE_BOOK.git'
     cd phone_book
     npm install
     npm start

Next step is Back-End.
